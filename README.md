# To-Do List App

## Description

This is a simple To-Do List application built with HTML, CSS, and JavaScript that allows you to manage your daily tasks efficiently. It features a user-friendly interface for adding, deleting, editing, and marking tasks as complete. The app also includes a progress bar to track your task completion rate.

## Features

- **Add tasks:** Easily add new tasks to your list with a simple input field.
- **Delete tasks:** Remove any unwanted tasks from your list.
- **Edit tasks:** Modify existing tasks as needed.
- **Mark as complete:** Toggle the completion status of tasks using checkboxes.
- **Progress bar:** Visualize your task completion progress with a dynamic progress bar.
- **Filter tasks:** View all tasks, completed tasks, or uncompleted tasks using the filter buttons.
- **Clear all tasks:** Delete all tasks from your list.
- **Local storage:** Persist your task list between browser sessions using local storage.
- **Confetti animation:** Celebrate completing all your tasks with a fun confetti effect.
- **Sound effect:** Enjoy a satisfying sound when you mark a task as complete.
- **Responsive design:** Adapts to different screen sizes for optimal viewing.

## Technologies Used

### HTML (index.html)

- **Semantic elements:** Uses elements like `header`, `main`, `ul`, `li`, etc. for better structure and SEO.
- **Form elements:** Utilizes `input`, `button` elements for user input.
- **Accessibility features:** Includes ARIA attributes and semantic HTML for screen readers.

### CSS (styles.css)

- **Flexbox:** Uses flexbox for layout and responsiveness.
- **CSS variables:** Defines reusable styles with variables for maintainability.
- **Media queries:** Implements a responsive design for different screen sizes.

### JavaScript (script.js)

- **DOM Manipulation:**  Dynamically updates the HTML content based on user interactions.
- **Event handling:** Responds to user events like clicks, input changes, etc.
- **Local Storage:** Persists data locally in the user's browser for data persistence.
- **Confetti and Sound Effects:** Integrates confetti animation and sound effects for user engagement.

## How to Use

1. **Adding a Task:**
   - Type your task in the input field.
   - Press "Enter" or click the "+" button to add the task to your list.
2. **Marking a Task as Complete:**
   - Click the checkbox next to a task to mark it as complete.
   - The task will be crossed out, and the progress bar will update accordingly.
3. **Editing a Task:**
   - Click the edit button (pencil icon) next to the task you want to edit.
   - Modify the task text directly.
   - Press "Enter" or click outside the task text to save the changes.
4. **Deleting a Task:**
   - Click the delete button (trash can icon) next to the task you want to remove.
5. **Filtering Tasks:**
   - Click the "All," "Done," or "Not Done" buttons to filter the displayed tasks.
6. **Clearing All Tasks:**
   - Click the "Clear All Tasks" button to delete all tasks. This action requires confirmation.
